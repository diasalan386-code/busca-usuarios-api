# 🌐 Busca de Usuários com API

Aplicação web que consome uma API pública para buscar e exibir usuários.

## 🚀 Funcionalidades

- Consumo de API com JavaScript
- Exibição dinâmica de dados
- Interface simples e responsiva

## 💻 Tecnologias

- HTML
- CSS
- JavaScript
- API REST

## 🎯 Objetivo

Praticar consumo de APIs, manipulação de dados e construção de interfaces dinâmicas.

## 🌐 API utilizada

https://jsonplaceholder.typicode.com/users
